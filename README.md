# Sparse Matrix Vector Multiplication

Sparse Matrix Vector Multiplication is a fundamental operation used in a widespread amount of applications.

This project aims to evaluate the performance of both sequential and parallel versions of SpMV under different combinations, using OpenMP and MPI for parallelization.
